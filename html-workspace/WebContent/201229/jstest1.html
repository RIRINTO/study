script에서 document 객체의 요소 ( body의 요소 ) 로 접근하기 위해서
	document.getElementById('id');  
	document.getElementsByTagName('Tag');
		: 배열을 리턴한다.
	document.getElementsByNamd('name');
		: 배열을 리턴한다.
	document.getElementsByClassName('class');
		: 배열을 리턴한다.

	ECMA2016, ECMA6
		: Script의 표준 번호
	document.querySelector();
		: 동일한 이름의 요소가 여러 개 있어도 첫 번째 요소만 접근한다.
	document.querySelectorAll();
		: 동일한 이름의 모든 요소에 접근, 배열을 리턴한다.
