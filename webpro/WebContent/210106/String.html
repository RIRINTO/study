charAt(index)
	: 지정된 인덱스 위치의 문자 하나를 가져온다.
charCodeAt(index)
	: 지정된 인덱스 위치에 위치하는 문자 하나의 유니코드 값을 가져온다.
		
trim()
	: DATA 앞뒤의 무효의 공백을 제거한다.
		
substring(from, to)
	: start ~ end 사이의 값을 추출한다.
slice(start, end)
	: start ~ end 사이의 값을 추출한다.
substr(start, length)
	: start부터 length의 갯수만큼 추출한다.
		
split("구분자", 갯수)
	: 구분자를 기준으로 문자열을 분리하여 배열로 저장한다.
