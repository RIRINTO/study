1. 입력하기 - prompt를 이용. return 값은 문자열 ( String ) 이다.
	var vname = prompt("이름을 입력하세요.", "한글로 입력");
	
2. div 요소에 출력하기 - id가 d1인 요소.
	var vd1 = document.getElementById('d1');
	
3. innerHTML or textContent를 이용하여 출력.
	내용 중 html 태그가 포함되어 있을 때는 innerHTML로 출력
	vd1.innerHTML = vname;
	내용 중 html 태그가 포함되어 있지 않을 때는 innerHTML 또는 textContent로 출력. 
	vd1.textContent = vname;
